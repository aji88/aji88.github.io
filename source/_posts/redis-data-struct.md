### Redis的数据类型
redis有9大数据类型，包括
* string  字符串
* list  哈希
* hash  列表
* set   集合
* zset  有序集合
* bitmap    位集合
* hyperLog  
* Geospatial Indexes 地理空间索引
* stream    流
